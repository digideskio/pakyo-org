With Pakyow, you start with the view and build everything else on top. It's easy:

1

Generate your app and immediately create views using HTML or your favorite template language.

2

Start the app and immediately use it in your local browser without writing any back-end code.

3

Add back-end code (written in Ruby) on top without changing any of the previously created views.
